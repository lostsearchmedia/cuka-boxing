# Cuka Boxing

Website for Cuka Boxing — deployed via Netlify at [cukaboxing.com](https://cukaboxing.com).

## Repository Structure

```
cuka-boxing/          ← repo root (upload your files here)
  index.html          ← main homepage (required)
  about.html          ← about page
  images/             ← all images folder
  README.md           ← this file
```

## How to Upload Your Files

1. Go to [github.com/lostsearchmedia/cuka-boxing](https://github.com/lostsearchmedia/cuka-boxing)
2. 2. Click **"uploading an existing file"**
   3. 3. Drag and drop the contents of your Cuka Boxing folder
      4. 4. Click **"Commit changes"**
        
         5. Netlify will automatically redeploy to cukaboxing.com after every commit to the `main` branch.
